# SPAE -- Sistema Profesional de Autoría de Evaluaciones

> **Una herramienta profesional para que docentes universitarios diseñen
> evaluaciones auténticas, rigurosas y pedagógicamente efectivas.**

**Versión:** 0.1.0 (Fundación)

## Propósito

SPAE es una aplicación web estática diseñada para ayudar al docente
universitario a crear, revisar, organizar y mejorar evaluaciones para
sus cursos, respetando siempre su autonomía académica.

El sistema **no reemplaza al docente**; ofrece asistencia basada en
principios pedagógicos, autenticidad profesional y buenas prácticas de
evaluación.

## Principios

-   El docente es el diseñador de la evaluación.
-   La herramienta sugiere; nunca impone decisiones.
-   Las preguntas responden a competencias y resultados de aprendizaje.
-   Se priorizan situaciones organizacionales auténticas.
-   Se utiliza terminología aceptada en Administración y Gestión en
    Latinoamérica.
-   Se procura reducir la sobrecarga cognitiva del docente.
-   Compatible con GitHub Pages.

## Público objetivo

Docentes universitarios de Administración, Gestión, RR.HH., Marketing,
Operaciones, Liderazgo, Emprendimiento y disciplinas afines.

## Tecnologías

-   HTML5
-   CSS3
-   JavaScript (ES6+)
-   JSON
-   LocalStorage

No requiere servidor ni base de datos.
